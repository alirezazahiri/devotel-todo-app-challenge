# Todo App Challenge

## Tech Stack

- Next.js
- Tailwind CSS (alongside shadcn/ui)
- TypeScript
- React Query
- React Hook Form
- Zod
- Redux Toolkit

Also using `dummyjson` as a mock API.

## Features

- [ ] Add a new todo
- [ ] Toggle todo completion status
- [ ] Delete a todo
- [ ] Edit a todo
- [ ] Filter todos by status (all, active, completed)
- [ ] Search todos by title (search input, case-insensitive)
